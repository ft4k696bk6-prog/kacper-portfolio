# Kacper Bernecki Portfolio

Nowoczesna strona portfolio / landing page Kacpra Berneckiego. Strona prezentuje praktyczne aplikacje webowe, CRM-y, automatyzacje, projekty, technologie oraz dane kontaktowe.

## Technologie

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Vercel
- Responsive Design

## Uruchomienie lokalne

```bash
npm install
npm run dev
```

Po uruchomieniu projekt będzie dostępny lokalnie pod adresem `http://localhost:3000`.

## Build produkcyjny

```bash
npm run build
```

## Lint

```bash
npm run lint
```

## Wdrożenie na Vercel

1. Utwórz nowe repozytorium na GitHubie.
2. Wypchnij kod do nowego repozytorium.
3. W Vercel wybierz `Add New Project`.
4. Podłącz repozytorium.
5. Framework powinien zostać rozpoznany jako Next.js.
6. Uruchom deploy.

Projekt nie wymaga zmiennych środowiskowych do podstawowego działania.
